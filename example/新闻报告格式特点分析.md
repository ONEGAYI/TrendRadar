# 新闻报告格式特点分析

## 📋 报告格式特点总结

### 🏗️ 整体结构框架

```
# 标题 - 日期
📊 报告概述
---
🏷️ 各大分类板块
---
📈 热度趋势分析
---
📋 总结
---
*免责声明*
```

#### 🎯 结构特点
- **层次清晰**: 采用标准Markdown标题层级（#、##、###）
- **逻辑性强**: 从概览到详细，再到分析总结的递进结构
- **分隔明确**: 使用 `---` 分隔各大板块，视觉上层次分明
- **标准化开头**: 每个报告都有统一的概述部分

---

## 📊 报告概述部分特点

### 核心信息展示
```markdown
## 📊 报告概述
- **新闻总数**: 100条
- **数据来源**: 微博、知乎、抖音
- **生成时间**: 2025年12月1日 16:08
- **时间范围**: 最近24小时内
```

#### 📝 特点分析
- **数据标准化**: 统一的信息展示格式
- **emoji图标系统**: 📊、📝、🕐等图标增强识别度
- **bullet points**: 使用列表形式，便于快速浏览
- **关键数据突出**: 加粗显示重要数值信息

---

## 🏷️ 分类板块结构特点

### 标准新闻条目格式
```markdown
## 🏷️ 大类标题
### 🎯 子分类标题

1. **新闻标题 - 评价总结**
   - 来源: [微博](链接) | [知乎](链接) | [抖音](链接)
   - 排名: 微博第1名，知乎第2名
   - 评价: 深度分析和专业观点
```

#### 🎨 视觉设计特点
- **双重标题系统**: 类别标题 + 具体新闻标题
- **emoji图标系统**: 每个分类都有对应的emoji图标
- **编号体系**: 按顺序编号，便于引用和查找
- **缩进层次**: 子信息使用缩进，体现信息层级

#### 📊 内容结构特点
1. **标题+评价格式**: "新闻标题 - 评价总结"
2. **多平台来源整合**: 同一新闻在不同平台的信息合并显示
3. **排名信息显示**: 各平台热度排名的详细对比
4. **深度评价分析**: 每条新闻都包含专业观点和分析

---

## 🔗 链接处理特点

### 链接展示格式
```markdown
- **单平台**: [微博](https://s.weibo.com/weibo?q=xxx)
- **多平台**: [微博](链接) | [知乎](链接) | [抖音](链接)
- **链接文本**: 使用平台名称作为链接文本，直观易懂
```

#### 🔧 技术特点
- **完整链接保留**: 所有新闻都包含原始来源链接
- **多平台整合**: 同一新闻在不同平台的链接都完整显示
- **链接文本友好**: 用平台名称作为链接文本，便于识别
- **分隔符使用**: 用 `|` 分隔不同平台链接，视觉清晰

## 📈 数据分析部分特点

### 热度分析结构
```markdown
### 🔥 今日热点排行
1. **国考相关话题** - 多个平台热搜，教育就业关注度最高
2. **香港大埔火灾** - 重大安全事故，引发广泛关注
...

### 📊 平台热度分布
- **微博热点**: 主要关注时事政治、娱乐八卦、社会事件
- **抖音热点**: 主要关注娱乐内容、生活方式、体育赛事
- **知乎热点**: 主要关注深度讨论、专业问题、社会现象分析

### 🎯 关键词云
- 教育: 国考、公务员考试、幼师招聘
- 娱乐: 疯狂动物城2、肖战、影视作品
...
```

#### 📊 分析特点
- **量化分析**: 具体的排名数据和分布统计
- **定性描述**: 对平台特点和用户偏好的深度分析
- **关键词提取**: 自动识别和分类热点关键词
- **趋势洞察**: 对热点现象的深度解读

## 📋 总结部分特点

### 总结内容结构
```markdown
今日新闻涵盖了社会生活的各个方面，从国内外的重大事件到日常生活的小确幸。值得关注的是：

1. **国考热度不减**，多个相关话题登上热搜...
2. **香港大埔火灾**造成重大人员伤亡...
3. **《疯狂动物城2》**持续热度不减...
...

整体来看，今日新闻既有严肃的社会议题，也有轻松的娱乐内容，反映了当前社会的多元化关注点。
```

#### 🎯 总结特点
- **数字化总结**: 用数字列出重点关注事件，便于记忆
- **平衡观点**: 既有严肃议题也有轻松内容，客观全面
- **社会洞察**: 对热点现象和社会趋势的深度分析
- **趋势判断**: 指出社会关注点的变化和发展方向
